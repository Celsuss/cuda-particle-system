# cuda-particle-system

## Build system
``` bash
    cmake .
    make
    ./cuda-particle-system
```

## Install cuda toolset
### Arch linux
Install cuda toolset.
``` bash
`sudo pacman -S cuda cuda-gbd nvidia-utils opencl-nvidia``

## Verify Your Installation
After installation, verify that CUDA is correctly set up:
``` bash
`nvcc --version``
